## Details

### API Frontends

Each API Frontend (HTTP, GraphQL, SPARQL, etc) have configuration options, some which are required.

For example, HTTP requires a mapping from resource URLs to the RDF data model, namely which subjects to query on a `/resource`.

Much of this can be inferred via the Schema Registry.

Additionally, GraphQL requires an optional mapping from GraphQL names to fully qualified IRIs.
